# Block Cell - SFS Pre-Origination Training Platform

Professional training platform for Square Financial Services (SFS) Pre-Origination compliance and credit review.

## 🌐 Live Site
**https://sabamahd.github.io/BlockCellTraining/**

## ✨ Features

### 🏠 Home Dashboard
- Welcome banner with quick start guide
- Progress tracking
- Latest updates and announcements
- Key statistics

### 🎓 New Hire Guide
- Complete 30-day onboarding timeline
- Week-by-week training schedule
- Interactive checklist
- Clear milestones and expectations

### 📚 Training Modules
- 8 comprehensive training modules
- Progress tracking for each module
- Estimated completion times
- Status badges (Completed/In Progress/Locked)

### 📖 Resources
- **SOPs**: All standard operating procedures
- **Refreshers & Quizzes**: 10+ refresher courses with quizzes
- **Research Tools**: Regulator, SOS search, AppOrchard
- **Video Tutorials**: Step-by-step guides
- **Quick Reference**: Checklists, flowcharts, FAQs

### 📅 Schedule
- New hire training sessions
- Refresher training calendar
- Upcoming deadlines
- Google Calendar integration

### 💬 Support & Contacts

## 🚀 Getting Started

### For New Hires
1. Start with the **New Hire Guide** tab
2. Review the 30-day timeline
3. Complete Week 1 modules before first live session
4. Check **Support & Contacts** for your team directory
5. Bookmark the **Resources** tab for quick reference

### For All Users
- **Resources** - Access all SOPs and refreshers
- **Schedule** - View upcoming training sessions
- **Training Games** - Practice your skills
- **Support** - Find who to contact for help
